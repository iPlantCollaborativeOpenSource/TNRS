Commands:

Taxamatch (taxamatch):
An algorithm for fuzzy (approximate string) matching of taxon scientific names. Select the source file you wish to compare your name against and the service will return the closest results.

Near Match (near_match):
This feature is used to calculate a generalized string to use for comparison in Taxamatch.

Normalize (normalize):
This is used to normalize a string.

Normalize Author (normalize_auth):
This is used to normalize an author.

 

This is an open source project and can be found at:
http://code.google.com/p/taxamatch-webservice
